# Flashcard Quiz App

A Java Swing application for studying using flashcards.

## Features
- Show Answer button
- Next and Previous navigation
- Add flashcards
- Edit flashcards
- Delete flashcards
- Simple GUI

## Technology Used
- Java
- Swing

## How to Run

Compile:
javac FlashcardQuizApp.java

Run:
java FlashcardQuizApp
